# HabitFlow — Трекер привычек

Мобильное веб-приложение для отслеживания привычек. Работает офлайн, без бэкенда — все данные хранятся в `localStorage`.

## Структура файлов

```
habitflow/
├── index.html   — разметка (HTML)
├── style.css    — стили (CSS)
├── app.js       — логика (JavaScript)
└── README.md
```

## Возможности

- ✅ Добавление привычек с иконкой, цветом, периодичностью
- 🔥 Серии (текущая и лучшая)
- 📊 Статистика: неделя (столбчатый график) и месяц (круговой)
- 🏆 9 достижений / значков
- 🚀 XP и уровни за каждое выполнение
- 📝 Заметки к каждому дню
- 🌙 Тёмная и светлая тема
- 🌐 Русский / English
- 🔔 Push-уведомления по расписанию
- 💾 Офлайн-хранение в localStorage
- 📤 Экспорт данных в JSON

## Запуск

Просто открой `index.html` в браузере — никаких зависимостей и сборки не нужно.

## Деплой на Vercel / Netlify

Залей папку целиком — проект статический, настройка не требуется.
